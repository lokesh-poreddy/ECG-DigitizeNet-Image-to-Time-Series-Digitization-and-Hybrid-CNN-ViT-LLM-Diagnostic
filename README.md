# ECG-DigitizeNet
Image-to-Time-Series Digitization and Hybrid CNN–ViT–LLM Diagnostic System.
## Features:
- Convert ECG images to 12-lead time-series.
- Hybrid CNN + Transformer classifier.
- Doctor Lokitrix LLM clinical interpretation.
